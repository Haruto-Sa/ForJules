# ForJules - AI パーソナルファッションアシスタント

AIを活用したパーソナルファッションアシスタントアプリケーション。ユーザーのワードローブを管理し、AI によるコーディネート提案を提供します。

## 🚀 プロジェクト概要

ForJules は以下の技術スタックで構築されています：

- **フロントエンド**: Next.js 14 + TypeScript + Tailwind CSS
- **バックエンド**: Go + Gorilla Mux + SQLite
- **AI/ML**: Python + OpenCV + scikit-learn + TensorFlow
- **データベース**: SQLite3
- **スタイリング**: Tailwind CSS + Heroicons

## 🛠️ セットアップ

### 前提条件
- Node.js 18.0.0 以上
- Go 1.21 以上
- Python 3.8 以上

### インストール手順

1. **リポジトリをクローン**
```bash
git clone <repository-url>
cd ForJules
```

2. **全依存関係を一括インストール**
```bash
npm run install-all
```

3. **データベース初期化**
```bash
npm run db:init
```

## 🚀 開発サーバー起動

### 個別起動
```bash
# フロントエンド開発サーバー（ポート3000）
npm run dev

# バックエンドサーバー（ポート8080）
npm run backend:dev

# Python ML スクリプト
npm run cvml:dev
```

### 動作確認
- フロントエンド: http://localhost:3000
- バックエンドAPI: http://localhost:8080/api/health
- API ドキュメント: 開発中

## 📁 プロジェクト構成

```
ForJules/
├── backend/           # Go バックエンド
│   ├── db/           # データベース関連
│   ├── handlers/     # HTTP ハンドラー
│   ├── models/       # データモデル
│   └── main.go       # エントリーポイント
├── frontend/         # Next.js フロントエンド
│   └── src/
│       ├── app/      # App Router ページ
│       ├── components/ # React コンポーネント
│       ├── lib/      # ユーティリティ
│       └── types/    # TypeScript 型定義
├── cvml/             # Python 画像解析・ML
├── docs/             # プロジェクトドキュメント
│   ├── finTask/      # 完了タスク記録
│   └── todo/         # タスク管理
└── package.json      # プロジェクト設定
```

## 📊 API エンドポイント

### ワードローブ管理
- `GET /api/wardrobe` - 全アイテム取得
- `POST /api/wardrobe` - アイテム作成
- `GET /api/wardrobe/{id}` - 特定アイテム取得
- `PUT /api/wardrobe/{id}` - アイテム更新
- `DELETE /api/wardrobe/{id}` - アイテム削除

### コーディネート提案
- `GET /api/outfit_suggestions` - 提案取得
- `POST /api/outfit_suggestions` - 提案生成

### システム
- `GET /api/health` - ヘルスチェック
