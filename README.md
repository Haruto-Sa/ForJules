# ForJules

## 使用方法

### バックエンドサーバーの起動

1. ターミナルで `backend` ディレクトリに移動します。
   ```bash
   cd backend
   ```
2. 以下のコマンドを実行して、Go Web API サーバーを起動します。
   ```bash
   go run main.go
   ```
   サーバーはデフォルトで `localhost:8080` で起動します。

### CV/ML (`cvml`) ディレクトリ

このディレクトリ (`cvml/`) には、Python を使用した画像解析や機械学習関連のコードが含まれる予定です。
- `analyze.py`: 画像から衣類の特徴を抽出するためのメインスクリプトです。

### フロントエンド (`frontend`) ディレクトリ

このディレクトリ (`frontend/`) には、Next.js やその他のフレームワークを使用したフロントエンドアプリケーションのコードが含まれる予定です。
- `src/`: 主なソースコードが格納されます。
- `public/`: 静的ファイル（HTML、画像など）が格納されます。
