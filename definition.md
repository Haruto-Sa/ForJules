# 要件定義

## 概要

Instagram 等の画像からコーディネートを解析し、ユーザーのワードローブと照合してパーソナライズされたコーディネート提案を行う Web アプリケーション。

## 機能要件

- Instagram 等の画像ソースからコーディネート画像を取得・解析
- 服の種類・色・パターン・スタイルタグ等のメタデータを持つワードローブ管理
- 画像解析による服の特徴抽出（コンピュータビジョン）
- ユーザーのワードローブと SNS コーディネートのマッチング
- パーソナライズされたコーディネート提案・ランキング
- 提案内容の保存・共有
- 直感的な UI/UX

## 非機能要件

- セキュアな認証・認可
- スケーラブルな構成
- モバイル対応
- 継続的なテストと改善

## 想定ユーザー

- ファッションに興味がある一般ユーザー
- 自分の服で新しいコーディネートを知りたい人

## 制約

- MCP サーバーは使用しない
- 主要言語は Go（バックエンド）、Python（CV/ML）、TypeScript/JavaScript（フロントエンド）
