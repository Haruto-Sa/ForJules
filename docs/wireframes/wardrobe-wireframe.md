# ワードローブ管理画面 ワイヤーフレーム

## 1. ページ概要

**目的**: ユーザーが所有する服を管理し、簡単に検索・編集できる画面

**主要機能**:
- アイテム一覧表示
- 検索・フィルタリング
- アイテム詳細表示
- 新規アイテム追加
- お気に入り管理

## 3. 機能詳細

### 3.1 検索機能
- **即席検索**: タイプ、色、ブランド名で検索
- **フィルター**: タイプ別、色別、季節別、スタイル別でフィルタリング
- **ソート**: 新しい順、お気に入り順、使用頻度順

### 3.2 アイテム表示
- **画像**: 正方形のサムネイル（実際の写真またはプレースホルダー）
- **基本情報**: タイプ、ブランド、色、パターン
- **アクション**: お気に入り切り替え、詳細表示、編集

### 3.3 お気に入り機能
- ハートアイコンでお気に入り状態を表示
- ワンクリックで切り替え可能
- お気に入りアイテムのみでフィルタリング可能

---

## 4. レスポンシブ対応

### モバイル（<768px）
- 2列グリッド表示
- 検索バーとフィルターを縦積み
- タブレット以上ではヘッダーにまとめて表示

### タブレット（768px-1024px）
- 3列グリッド表示
- フィルターボタンを横並び

### デスクトップ（>1024px）
- 4列グリッド表示
- 全ての要素を横並びで最適化

---

## 5. 状態管理

### 空の状態
```
┌─────────────────────────────────────────────────────────────────┐
│                           📷                                   │
│                   まだアイテムがありません                      │
│             最初のアイテムを追加してワードローブを作り始めましょう  │
│                   [+ アイテムを追加]                           │
└─────────────────────────────────────────────────────────────────┘
```

### ローディング状態
- スケルトンローディング表示
- アイテムカードの形状を維持したプレースホルダー

### エラー状態
- エラーメッセージと再試行ボタン表示
- ネットワークエラー時の適切な案内

---

## 6. インタラクション

### ホバー効果
- カードにシャドウ追加
- ボタンの色変化
- 滑らかなトランジション

### クリック/タップ
- アイテム画像クリック → 詳細ページ
- 編集ボタン → 編集画面
- お気に入りボタン → 即座に状態変更

### キーボードナビゲーション
- Tab キーでフォーカス移動
- Enter キーで選択実行
- アクセシビリティ準拠

---

## 7. 技術実装メモ

### 使用コンポーネント
- `WardrobeGrid`: アイテム一覧表示
- `ItemCard`: 個別アイテムカード
- `SearchFilter`: 検索・フィルター
- `LoadingSpinner`: ローディング表示

### 状態管理
- `useState`: ローカル状態（検索文字列、フィルター選択）
- `useEffect`: API データ取得
- カスタムフック: `useWardrobe` でAPIロジック分離

### パフォーマンス最適化
- 画像の遅延読み込み
- 仮想スクロール（大量アイテム対応）
- デバウンス検索
